# 03 - 运行与验证

这一节记录怎么在 IDEA 里跑起来，以及应该看什么。

## 启动方式

先设置环境变量：

```bash
set DASHSCOPE_API_KEY=你的key
set DASHSCOPE_MODEL_NAME=qwen-plus
set DASHSCOPE_TIMEOUT=PT5M
```

然后在项目根目录执行：

```bash
mvn spring-boot:run
```

## 你会看到什么

- Spring Boot 启动成功
- `OutlineNode` 生成提纲
- `DraftNode` 生成初稿
- `ReviewNode` 打分
- `ReviseNode` 修订
- `PublishNode` 发布
- 最后打印 `Final snapshot next node: __END__`

## 验证重点

1. `messages` 是否在追加
2. `score` 是否会从低分变成高分
3. `revisionCount` 是否会增长
4. checkpoint 是否能暂停并恢复
5. 最终文档是否写回 state
