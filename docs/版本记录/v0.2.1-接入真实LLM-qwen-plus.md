# v0.2.1 接入真实 LLM（qwen-plus）

## 目标

把工作流里的 toy 模型替换成真实的 Qwen 模型，让项目可以直接调用在线 LLM。

## 内容

- 增加 `langchain4j-open-ai`
- 使用 DashScope OpenAI 兼容接口
- 模型名固定为 `qwen-plus`
- 通过环境变量读取 `DASHSCOPE_API_KEY`
- 增加更长的请求超时，避免学习案例在首轮请求里过早超时
- 保留原有工作流、条件分支、循环和 checkpoint

## 验证结果

- 项目可编译
- Spring Boot 可启动
- `outline -> draft -> review -> revise -> publish` 能完整执行
- checkpoint 能暂停到 `publish` 前并恢复

## 启动方式

```bash
set DASHSCOPE_API_KEY=你的key
set DASHSCOPE_MODEL_NAME=qwen-plus
set DASHSCOPE_TIMEOUT=PT5M
mvn spring-boot:run
```
